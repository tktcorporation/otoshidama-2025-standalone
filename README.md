## 📖 概要

お年玉ガチャは、お年玉の金額をランダムに決定できる楽しいWebアプリケーションです。新年の贈り物をより楽しく、エキサイティングにする新しい方法を提供します。

### ✨ 主な機能

- 🎯 お年玉の金額をガチャで決定
  - アニメーション付きのガチャ演出
